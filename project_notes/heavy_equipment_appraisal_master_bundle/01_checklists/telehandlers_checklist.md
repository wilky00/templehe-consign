# Telehandlers Appraisal Checklist

Complete the shared intake first, then use this category checklist.

## Quick ID / Configuration

- Open station / canopy / enclosed cab
- 2WD / 4WD / track style if applicable
- Emissions label visible / not visible
- Running / partially running / non-running
- Hours displayed and whether believed accurate

## Major Components to Score (0 to 5)

- Engine
- Transmission
- Boom assembly
- Hydraulic system
- Axles/steering
- Tires
- Cab/controls
- Attachments/stabilizers

## Inspection Prompts

- Cold start
- Boom section wear
- Boom extend/retract smoothness
- Frame leveling operation
- Stabilizer function
- Hydraulic leaks
- Tire wear
- Brake performance
- Attachment lock-up
- Cab condition

## Common Attachments / Options to Capture

- Fork carriage
- Bucket
- Jib
- Winch
- Personnel basket
- Truss boom
- Work lights

## Required Photos

- 4 corner exterior
- Serial plate
- Hour meter
- Cab interior
- Engine compartment
- Boom wear pads
- Fork carriage or bucket
- Tires
- Outriggers
- Visible leaks/damage

## Key Value Drivers

- Hours
- Lift height/capacity
- Enclosed cab vs open
- Attachment package
- Boom wear
- Tire life
- Outrigger/frame leveling condition
- Brand demand
- Running status
- ANSI compliance features

## Red Flags

- Structural cracks, poor weld repairs, or missing guards
- Major active hydraulic, engine, coolant, or drivetrain leaks
- Meter inconsistency or suspected hour tampering
- Serial plate missing or unreadable
- Safety systems inoperative
- Major component failure, non-running status, or inability to demonstrate core functions
- Attachment mismatch, missing coupler parts, or significant bucket/blade/drum/body wear

## Appraiser Notes
- Cosmetics summary
- Immediate repair needs
- Resale readiness
- Consignment suitability
- Direct-buy suitability
