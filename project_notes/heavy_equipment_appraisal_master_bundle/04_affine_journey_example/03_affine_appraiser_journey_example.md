# Appraiser Journey Example: Excavator Appraisal Flow

This example shows one simple future-state flow for an excavator appraisal. It is written so it can be imported as a page into AFFiNE.

## 1. Arrive on Site
- Confirm customer contact
- Verify the machine to be appraised
- Walk around for first impressions
- Check whether the unit appears safe to inspect and operate

## 2. Start Shared Intake
- Create appraisal record
- Capture customer, site, make, model, serial, year, and hours
- Choose category: Excavators
- Mark running status
- Record cab type, emission label, ownership, and service-record availability

## 3. Capture Initial Photos
- 4-corner exterior
- Serial plate
- Hour meter
- Cab interior
- Engine compartment

## 4. Configure the Excavator Branch
- Capture bucket type
- Capture coupler
- Capture thumb
- Capture auxiliary hydraulics
- Capture any extra attachments included in sale

## 5. Functional Inspection
- Observe cold start if possible
- Note smoke, leaks, abnormal noise
- Test boom, stick, bucket, swing, and travel functions
- Check swing bearing play
- Check track and undercarriage wear
- Inspect pins, bushings, and visible weld repairs

## 6. Score Major Components
- Engine
- Hydraulic pumps/system
- Swing system
- Boom/stick/bucket
- Undercarriage
- Travel motors/drives
- Cab/controls
- Attachments

## 7. Complete Required Category Photos
- Boom base
- Stick and bucket
- Coupler/thumb
- Undercarriage both sides
- Visible leaks or damage

## 8. Review Red Flags
- Structural damage
- Major active leaks
- Missing serial plate
- Non-running or partial function
- Unsafe or inoperative controls
- Heavy undercarriage cost exposure

## 9. Generate Summary
- Overall weighted score
- Cosmetic condition
- Marketability rating
- Immediate repair exposure
- Resale notes
- Consignment suitability
- Direct-buy suitability

## 10. Submit Appraisal
- Confirm required fields complete
- Confirm required photos complete
- Save and submit for review or listing preparation
